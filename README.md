# Brand Sentiment Analysis

This project is a simple sentiment analysis dashboard built using Python and Streamlit.

The application analyzes airline customer reviews and classifies them as Positive, Negative, or Neutral.  
It also highlights common customer problems such as delays, cancellations, service issues, and refund-related complaints.

## Features
- Sentiment analysis of customer reviews
- Airline-wise filtering
- Identification of common customer problems
- Live sentiment prediction for user-entered reviews
- Simple and interactive dashboard

## Tools and Libraries Used
- Python
- Streamlit
- Pandas
- VADER Sentiment Analyzer
- Plotly
